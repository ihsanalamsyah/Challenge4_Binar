# Challenge4_Binar

Dalam challenge ini lanjutan dari challenge sebelumnya, saya menambahkan sebuah game berupa suit jepang dengan menggunakan bahasa JavaScript

dalam menjalankan suit jepang ini harus memilih antara gunting, kertas, dan batu
computer akan mengeluarkan gunting, kertas, atau batu secara acak

dibuat pengkondisian jika player memilih gunting dan computer memilih kertas player menang
dan jika player memilih kertas dan computer memilih gunting, computer menang dst.
